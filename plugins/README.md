# plugins/

Estensione user-side dello stack.

Vedi [docs/PLUGINS.md](../docs/PLUGINS.md) per la guida step-by-step.

Cartelle:
- `example-mcp/` — scheletro MCP server (FastMCP streamable-http)

Per il tuo plugin, copia uno dei due e modifica.

Plugin community: apri una PR per linkare il tuo nella tabella di [docs/PLUGINS.md](../docs/PLUGINS.md).
